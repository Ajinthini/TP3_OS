# TP3 Système d'Exploitation - Biceps v3

## Structure du projet
* **biceps.c** : Interpréteur de commandes (Readline).
* **creme.c** : Gestion des threads et de la liste chaînée avec Mutex.

## Compilation
* `make` : Génère le binaire `biceps`.
* `make memory-leak` : Génère le binaire pour Valgrind.
* `make clean` : Nettoie le répertoire.
